CF4M

Client Framework for Minecraft

Quick Start →

Easy

No need to write Event,Module and Setting etc.

Small

Less than 40KB.

Lightweight

Only used Guava libraries (Minecraft comes with.

Add it in your root build.gradle at the end of repositories

allprojects {
	repositories {
		maven { url 'https://cf4m.github.io/maven' }
	}
}

Add the dependency

dependencies {
	implementation 'cn.enaium.cf4m:cf4m:LATEST'
}
